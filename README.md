## рџ› пёЏ Mini-Projects Built

Here are the mini-projects implemented so far during my learning journey:
* **Calculator** ([calculator.py](file:///E:/GitHub/PYTHON-JOURNEY/mini_projects/calculator/calculator.py)): A basic CLI calculator handling addition, subtraction, multiplication, and division.
* **Weight Converter** ([weight_converter.py](file:///E:/GitHub/PYTHON-JOURNEY/mini_projects/weight_converter/weight_converter.py)): Converts input weight between Kilograms (Kgs) and Pounds (Lbs).
* **Temperature Converter** ([temp_converter.py](file:///E:/GitHub/PYTHON-JOURNEY/mini_projects/temp_converter/temp_converter.py)): Converts temperatures between Celsius and Fahrenheit.
* **Compound Interest Calculator** ([compound_interest_calculator.py](file:///E:/GitHub/PYTHON-JOURNEY/mini_projects/compound_interest_calculator/compound_interest_calculator.py)): Computes compound interest based on validated user inputs (principle, rate, time).
* **Countdown Timer** ([countdown_timer.py](file:///E:/GitHub/PYTHON-JOURNEY/mini_projects/countdown_timer/countdown_timer.py)): Uses Python's `time` library to create a tick-by-tick console countdown.
* **Shopping Cart** ([shopping_cart.py](file:///E:/GitHub/PYTHON-JOURNEY/mini_projects/shopping_cart/shopping_cart.py) & [shopping_cart_basic.py](file:///E:/GitHub/PYTHON-JOURNEY/mini_projects/shopping_cart/shopping_cart_basic.py)): Interactive scripts tracking selected item names, quantities, and final prices.
* **Madlibs Game** ([madlibs.py](file:///E:/GitHub/PYTHON-JOURNEY/mini_projects/madlibs/madlibs.py)): A simple word game that inserts user inputs (adjectives, nouns, verbs) into a humorous story template.
* **Quiz Game** ([quiz_game.py](file:///E:/GitHub/PYTHON-JOURNEY/mini_projects/quiz_game/quiz_game.py)): A CLI multiple-choice quiz testing knowledge with basic score tracking.
