## 🔍 Topics Covered

### 1. Variables & Typecasting
* **Concepts:** Declaring, naming constraints, checking type signatures. Converting strings to integers/floats/booleans for computations.
* **Syntax:**
  ```python
  age = 21                      # Int
  gpa = 3.9                     # Float
  name = "Jishnu"               # Str
  is_student = True             # Bool
  
  # Typecasting
  float_age = float(age)
  str_gpa = str(gpa)
  ```
* **Associated Files:** [03_Variable.py](file:///E:/GitHub/PYTHON-JOURNEY/BROCODE/03_Variable.py), [04_Typecasting.py](file:///E:/GitHub/PYTHON-JOURNEY/BROCODE/04_Typecasting.py)

### 2. User Input & Math Operations
* **Concepts:** Fetching strings from standard input, parsing numeric values, implementing mathematical functions, and utilizing the built-in `math` module.
* **Syntax:**
  ```python
  name = input("Enter name: ")
  val = float(input("Enter number: "))
  
  import math
  hypotenuse = math.sqrt(pow(a, 2) + pow(b, 2))
  ceil_val = math.ceil(9.1)
  ```
* **Associated Files:** [05_UserInput.py](file:///E:/GitHub/PYTHON-JOURNEY/BROCODE/05_UserInput.py) - [03.py](file:///E:/GitHub/PYTHON-JOURNEY/BROCODE/05_UserInput03.py), [06_ArithmeticsAndMaths.py](file:///E:/GitHub/PYTHON-JOURNEY/BROCODE/06_ArithmeticsAndMaths.py) - [04.py](file:///E:/GitHub/PYTHON-JOURNEY/BROCODE/06_ArithmeticsAndMaths04.py)

### 3. Conditional Flow & Logical Operators
* **Concepts:** Standard conditional branching, checking multiple conditions with logical operators, and writing concise inline conditional expressions.
* **Syntax:**
  ```python
  # Conditional & Logicals
  if age >= 18 and has_license:
      print("Eligible to drive")
  elif age < 18 or not has_license:
      print("Ineligible")
  
  # Ternary operator
  status = "Adult" if age >= 18 else "Minor"
  ```
* **Associated Files:** [07_IfStatementents.py](file:///E:/GitHub/PYTHON-JOURNEY/BROCODE/07_IfStatementents.py) - [01.py](file:///E:/GitHub/PYTHON-JOURNEY/BROCODE/07_IfStatementents01.py), [11_LogicalOperators.py](file:///E:/GitHub/PYTHON-JOURNEY/BROCODE/11_LogicalOperators.py) - [01.py](file:///E:/GitHub/PYTHON-JOURNEY/BROCODE/11_LogicalOperators01.py), [12_ConditionalExpression.py](file:///E:/GitHub/PYTHON-JOURNEY/BROCODE/12_ConditionalExpression.py)

### 4. String Slicing & Format Specifiers
* **Concepts:** Accessing characters, creating custom substrings via start/stop/step parameters, validating format properties, and structuring f-string outputs.
* **Syntax:**
  ```python
  # String Slicing
  first_name = name[0:4]
  reversed_name = name[::-1]
  
  # Format Specifiers
  print(f"Price: ${price:.2f}")  # 2 decimal places
  print(f"Name: {name:>10}")    # Right alignment
  ```
* **Associated Files:** [13_StringMethods.py](file:///E:/GitHub/PYTHON-JOURNEY/BROCODE/13_StringMethods.py) - [01.py](file:///E:/GitHub/PYTHON-JOURNEY/BROCODE/13_StringMethods01.py), [14_StringIndexing.py](file:///E:/GitHub/PYTHON-JOURNEY/BROCODE/14_StringIndexing.py), [15_FormatSpecifiers.py](file:///E:/GitHub/PYTHON-JOURNEY/BROCODE/15_FormatSpecifiers.py)

### 5. Loop Structures
* **Concepts:** Infinite validation loops, iterating through numbers/characters/sequences, using `break` and `continue` keywords, and nesting loops for multi-dimensional operations.
* **Syntax:**
  ```python
  # While loop validation
  while len(username) == 0:
      username = input("Cannot be blank. Enter username: ")
      
  # For loop with range
  for i in range(1, 11, 2):
      if i == 5:
          continue
      print(i)
  ```
* **Associated Files:** [16_WhileLoops.py](file:///E:/GitHub/PYTHON-JOURNEY/BROCODE/16_WhileLoops.py) - [02.py](file:///E:/GitHub/PYTHON-JOURNEY/BROCODE/16_WhileLoops02.py), [18_ForLoops.py](file:///E:/GitHub/PYTHON-JOURNEY/BROCODE/18_ForLoops.py), [20_NestedLoop.py](file:///E:/GitHub/PYTHON-JOURNEY/BROCODE/20_NestedLoop.py) - [01.py](file:///E:/GitHub/PYTHON-JOURNEY/BROCODE/20_NestedLoop01.py)

### 6. Collections: Lists & Sets
* **Concepts:** Storing multiple values in collections. Understanding mutability and ordering properties.
* **Syntax:**
  ```python
  # Lists: Ordered, changeable, duplicates OK
  fruits = ["apple", "orange", "banana"]
  fruits.append("pineapple")
  fruits.sort()
  
  # Sets: Unordered, immutable values (add/remove OK), no duplicates
  colors = {"red", "green", "blue"}
  colors.add("yellow")
  ```
* **Associated Files:** [21_ListSetTuple.py](file:///E:/GitHub/PYTHON-JOURNEY/BROCODE/21_ListSetTuple.py), [21_ListSetTuple01.py](file:///E:/GitHub/PYTHON-JOURNEY/BROCODE/21_ListSetTuple01.py)
